#littlepwgen
*Very simple password generator*


###What do you need before compiling
```bash
sudo apt-get install g++ libboost-program-options libboost-program-options-dev
```

###Installation
```bash
cd littlepwgen &&
mkdir out && 
cd out/ &&
cmake ../ &&
and make
```

###Usage
```bash
littlepwgen --help
```
